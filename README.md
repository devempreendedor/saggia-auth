<h1>Saggia Auth Api</h1>

<p>
   An authentication api for a personal project
</p>

## Installation

1. Clone the repo

```sh
git clone https://github.com/devempreendedor/saggia-auth.git
```

2. Install NPM packages

```sh
npm install
```

3. to run the unit tests and the integration tests

```sh
npm test
```
